Software & Tools

Open-source tools developed by our lab

ChIP-seq Analysis Pipeline

High-performance pipeline for ChIP-exo and ChIP-seq data analysis. Optimized for transcriptional regulatory network reconstruction in bacteria.

Python NGS Bacteria
GitHub Docs

Enzyme Cofactor Predictor

Deep learning-based tool for NAD/NADP cofactor prediction using transformer attention analysis. Enables rational enzyme engineering.

Deep Learning Transformer Enzyme
GitHub Paper

Battery SoH Estimator

Transformer-based framework for configuration-agnostic Li-ion battery State of Health estimation. Supports various battery configurations.

ML Battery Transformer

Genome-Scale Model Tools

Tools for genome-scale metabolic model reconstruction and analysis. Includes flux balance analysis and pathway optimization modules.
